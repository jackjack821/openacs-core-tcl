OpenACS Release Notes

Version 5.0.0

by Don Baccus
OpenACS docs are written by the named authors, and may be edited by OpenACS documentation staff.

This is a pre-alpha release of OpenACS 5.0.0.

Please report bugs using our Bug Tracker at the OpenACS website.

You may want to begin by reading our installation documentation for Chapter 4. Note that the Windows documentation is not current for OpenACS 4.7.0d, but an alternative is to use John Sequeira's Oasis VM project.

After installation, the full documentation set can be found by visiting http://[your-host]/doc.
